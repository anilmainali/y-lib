1 /* vugen.h - A.U. Luyer - 2012-03-27
2  * Last update 2014-01-09
3  * This header file contains the most used 'standard C functions' in Vugen scripts.
4  * By declaring these functions properly the compiler can check if these functions
5  * are used correctly and apply implicit conversions when necessary.
6  * This header file 'de-strips' the compiler in Vugen.
7  * It contains all functions mentioned in the HP LoadRunner Online Function Reference.
8  * NOTE: Sites like http://www.cplusplus.com/reference/clibrary/ explain the standard C functions
9  * better than the HP LoadRunner Online Function Reference.
10  *
11  * Usage: add vugen.h to the script and add #include "vugen.h" to globals.h.
12  * Or: add to include directory and add the include to lhrun.h on all Vugen machines and load generators.
13  */

177 /***** locale.h functions *****/
178 /* BEWARE: Vugen executes a setlocale(LC_ALL, "") before the start -- so the system's default locale
179  * is active instead of the minimal locale and the default for all C compilers: setlocale(LC_ALL, "C").
180  * This may cause functions like sscanf to behave unexpectedly, for instance when the decimal point
181  * is a comma (as in many European languages) sscanf(buf , "%lf", &double_var) will not yield the correct
182  * number.
183  * To print the currently active locale use: lr_log_message("%s", setlocale(LC_ALL, NULL));
184  */
